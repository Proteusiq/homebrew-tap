# Proteusiq's Homebrew Tap

Homebrew formulas for tools maintained by [@Proteusiq](https://github.com/Proteusiq).

## Usage

```bash
brew install proteusiq/tap/<formula>
```

## Available formulas

- [`ona`](https://github.com/Proteusiq/ona) — Beautiful markdown to HTML renderer that opens in your browser
  ```bash
  brew install proteusiq/tap/ona
  ```

## How it works

Formulas in `Formula/` are automatically updated by each project's release
pipeline (via [`dist`](https://github.com/axodotdev/cargo-dist)) whenever a
new version is tagged.
